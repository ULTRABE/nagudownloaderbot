# Telegram Video Downloader Bot

A fast Telegram bot that downloads and sends videos from popular platforms directly in chat.

## Supported Platforms

- YouTube (videos & shorts)
- Instagram Reels
- Facebook videos & shorts
- Twitter (X) videos
- Pinterest shorts

## Features

- Works in private chats and groups
- Automatic short vs long video detection
- High-quality shorts with small file size
- Fast download and upload
- Auto-deletes link messages
- Pins sent videos automatically
